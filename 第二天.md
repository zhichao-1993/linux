## 14.2基础命令(目录相关命令)
 + 主目录：HOME
   + root:/root
   + 普通用户：/home/username
   + ~:用户主目录
   
 + cd(切换目录)
   + cd 或 cd ~:回到当前用户的主目录
   + cd ~username:回到指定用户的主目录
   + cd -:在上一个目录和当前目录之间来回切换
   + cd!$:将上一条命令的参数作为cd的参数
   + .:当前目录
   + ..:上一级目录
   + 相关环境变量
     + echo $PWD:保存当前目录路径
     + ehco $OLDPWD:上一次所在目录路径，相当于cd -
     
  + pwd:显示当前目录路径
  
  + ls（文件处理命令）
    + ls 选项 参数
      + -a,--all :显示所有文件包含隐藏文件
      + -l:长格式，显示文件详细信息
      + -d:显示目录的详细信息
      + -i:查看目录下文件的i节点
      + -h:单位换算
      + -r:逆序显示
      + -R:递归显示
      
  + 文件类型
    + -:文件
    + d：目录
    + l:软连接
    
  + 文件权限
    + r:读
    + w:写
    + x:执行
